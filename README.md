#  Disable Windows Defender

**Disable Windows Defender** is a simple and portable utility that allows you to quickly disable Windows Defender Real-Time Protection, Tamper Protection, and other security components when needed for software installation, troubleshooting, or compatibility reasons.

## Key Features
- One-click disable of Windows Defender
- Disables Real-Time Protection, Cloud Protection, and Tamper Protection
- Portable — no installation required
- Option to re-enable protection
- Clean and simple interface

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive.
3. **Run** `DisableWindowsDefender.exe` as Administrator.
4. Choose the desired action (Disable or Enable).
5. Restart your computer if prompted.

**Strong Recommendation:** Re-enable Windows Defender after completing your task.
